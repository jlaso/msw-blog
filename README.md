# msw-blog
A simple blog for MySimpleWeb

Views are based on AdminLTE

you can install into web/custom this way
-> git submodule add -b master -f --name adminlte https://github.com/almasaeed2010/AdminLTE.git web/custom/adminlte

if your project is based on my-simple-web, the new versions (>1.0.0) incorpores AdminLTE by default
